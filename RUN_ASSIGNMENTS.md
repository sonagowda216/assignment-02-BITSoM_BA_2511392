## **Troubleshooting**

### If script doesn't run:
1. Ensure Python 3.7+ is installed: `python --version`
2. Check you're in the correct directory
3. Try running with explicit encoding: `set PYTHIOENCODING=utf-8 && python script_name.py`

### If output looks garbled:
- ✓ Fixed! All Unicode characters have been replaced with ASCII alternatives
- Try updating your terminal/PowerShell to the latest version
